# Simplify Skill

Launch the code-simplifier agent for code clarity and maintainability.

## Usage

```
/simplify [optional: specific files or feature]
```

## What this does

Spawns the code-simplifier agent to:
- Simplify complex code
- Improve readability
- Remove redundancy
- Ensure consistency
- Preserve functionality

## Implementation

```yaml
type: agent
agent: code-simplifier:code-simplifier
autorun: true
```
